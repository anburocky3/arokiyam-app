# Arokiyam - Open Source Desktop Wellness App

Arokiyam is a free, cross-platform desktop application designed to protect your physical and mental health while you work. By actively tracking screen time, hydration, and estimating stress based on typing patterns, Arokiyam ensures you are taking adequate rests and staying focused. 

## Features
- **Smart Stress Tracking**: Analyzes local typing behavior safely to measure stress limits.
- **Micro-Breaks & Eye Tests**: Recommends breaks without intruding on your workspace.
- **Hydration Reminders**: Keeps your body's energy up.
- **Privacy First**: 100% locally evaluated with no data leaving your machine.
- **Cross-Platform Support**: Built natively for Windows, macOS, and Linux.

## Tech Stack
### App
- **Desktop Framework**: Electron
- **UI Framework**: React
- **Styling**: Tailwind CSS / Vanilla CSS
- **Packaging**: Electron Builder

### Landing Page
- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: TailwindCSS

## Getting Started (Web)
To run the landing page locally:
```bash
npm install
npm run dev
```
